How to Upload the Website

Steps to put your website online

1
Download all files from the ZIP/folder I sent you

You should have: index.html, services.html, shop.html, about.html, branches.html, appointment.html, blog.html, shared.js, shared.css, Code.gs

2
Upload all .html, shared.js, and shared.css files to your hosting

All files must be in the SAME folder (root directory). Upload using your hosting's File Manager or FTP (FileZilla). Do NOT put them inside sub-folders.

3
Create the team photos folder

In your hosting file manager, create folder: assets/team/ — this is where you upload team photos later.

4
Open your website in the browser and test

Press Ctrl + Shift + R (Windows) or Cmd + Shift + R (Mac) to clear cache. Check all navigation links work.

✅ After uploading: Set index.html as your homepage in your domain settings (most hosting does this automatically).

Connecting Google Sheets (App Script Setup)

This is how customer enquiries and appointments arrive in your Google Sheet

⚠️ You need to do this once. After setup, all enquiries and bookings go automatically to your Google Sheet.
1
Open your Google Sheet

Go to sheets.google.com → open your Bhavana sheet (or create a new one named "Bhavana Clinic").

2
Open Apps Script

Click Extensions in the menu → click Apps Script. A new tab opens.

3
Paste the Code.gs file

In Apps Script, delete everything that's already there. Open your Code.gs file → copy everything → paste it into Apps Script.

4
Save the script

Press Ctrl + S (or click the 💾 save icon). Name the project "Bhavana Website".

5
Deploy as a Web App

Click Deploy (blue button, top right) → New deployment.
Set: Type = Web App
Set: Execute as = Me
Set: Who has access = Anyone
Click Deploy.

6
Copy the Web App URL

After deploying, you will see a URL like:
https://script.google.com/macros/s/ABC123.../exec
Copy this URL — you need it in the next step.

7
Paste the URL into shared.js

Open shared.js in Notepad (or any text editor). Find this line near the top:

var SHEET_URL = "https://script.google.com/macros/s/YOUR_URL_HERE/exec";

Replace YOUR_URL_HERE with your actual URL. Save the file. Re-upload shared.js to your website.

8
Test it works

Open the Web App URL in your browser. You should see:
{"status":"ok","message":"Bhavana Sheet API running ✅"}
If you see this, it is working!

✅ Tip: If you update Code.gs later, you must deploy a New deployment again (not Manage existing). The old URL still works — use the new URL if you want to update.

How to Update Content

All your data is in shared.js — easy to find and change

📌 Open shared.js in any text editor (Notepad, VS Code). Use Ctrl + F to search for what you want to change.

To change a product price:

Search for the product name → find p: → change the number.

{ id:3, name:"Signia Pure 312 3X", brand:"Signia", p:35000, emi:"EMI: ₹2,917/month" ... }

To add a new product:

Find BHAVANA.products → add a new line at the end of the array following the same pattern.

To change phone number:

Search for 08197428456 in shared.js — replace all occurrences.

To change WhatsApp number:

Search for 918197428456 in shared.js — replace all occurrences.

To update the Google Sheet URL:

Search for SHEET_URL at the top of shared.js → update the URL.

var SHEET_URL = "https://script.google.com/macros/s/YOUR_URL/exec";
✅ After editing shared.js, save the file and re-upload it to your website. Press Ctrl+Shift+R in the browser to clear the cache and see the changes.

Troubleshooting Common Problems

What to do if something doesn't work

❓ Enquiry form is not sending to Google Sheet

1

Open shared.js → find SHEET_URL → check it has the correct URL (from Step 6 of the Google Sheet setup).

2

Open the URL in your browser — it should show {"status":"ok"...}. If you get an error, re-deploy the Apps Script.

3

In Apps Script: Deploy → Manage deployments → check it still says "Active".

❓ Website looks broken / styles missing

Make sure shared.css and shared.js are in the SAME folder as all .html files.

❓ Team photos not showing

Check: (1) Folder is assets/team/ (2) File name matches exactly (lowercase, hyphens) (3) File is JPG format.

❓ Nav links go to wrong page

All pages use shared.js for the nav. Make sure the latest shared.js is uploaded.

❓ Need to add a new page

Send me the content and I will create the page for you — it takes about 30 minutes to add a new page to the website.
